Папка Config Program: программа конфигурации. Вход защищен паролем, пароль читается из бинарного файла. Сама программа задает путь к логу для кейлоггера, папку для скринов, интервал для скринов и e-mail, на который нужно отправлять логи
Папка Perform Program: Избинарного файла считывает конфигурацию, запускает кейлоггер и скриншотер, пишет логи, сохраняет изображения, прописывает себя в реестр для автозагрузки. 

На данный момент не реализовано: отправка на e-mail, немного нужно еще повозиться с кейлоггером (фишка его в том, что event возникает при отпускании клавиши => нельзя определить, в каком регистре напечатана буква, т.к. Shift сработает только при отпускании), немного переделать автозагрузку.
